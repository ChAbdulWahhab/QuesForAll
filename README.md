# QuesForAll 🎯

QuesForAll is a simple and interactive quiz web application where users can test their knowledge across multiple categories. The platform provides multiple-choice questions from different topics, making learning fun and engaging.

## 🌐 Live Demo

https://quesforall.netlify.app/

## 📚 Categories

The quiz includes questions from several categories such as:

* General Knowledge
* Sports
* Movies
* Music
* History
* Science
* Technology
* And more

## ✨ Features

* Multiple quiz categories
* Interactive multiple-choice questions
* Instant answer selection
* Clean and responsive UI
* Fast and lightweight website
* Works on desktop and mobile devices

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Netlify (for deployment)

## 🚀 How to Run Locally

1. Clone the repository

```
git clone https://github.com/yourusername/quesforall.git
```

2. Open the project folder

3. Run the project by opening `index.html` in your browser.
